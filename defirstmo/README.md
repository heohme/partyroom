# defirstmo



## 开始

``` sh
$ npm i @xhs/formula-cli@latest -g  # 全局安装构建工具
$ npm install                       # 安装依赖
$ npm run dev                       # 启动项目
```

> `npm run dev` 在首次执行的时候会自动升级 latest 包为最新版本，所以需要执行两次
